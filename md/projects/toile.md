## Toile

A clean, easy-to-use CRUD application for showcasing artistic works — built with a minimalistic and elegant design philosophy.

### Tech Stack

- **React.js** — component-driven UI with smooth interactions
- **GSAP** — animations and page transitions
- **Tailwind CSS** — utility-first styling for a responsive layout
- **Appwrite** — backend for CRUD operations and seamless user authentication

### Features

- Create, read, update, and delete artistic entries
- Smooth, performant animations powered by GSAP
- Secure user authentication via Appwrite
- Fully responsive across mobile and desktop
